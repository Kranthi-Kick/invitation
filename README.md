# Wedding RSVP Website

A beautiful wedding RSVP website with Supabase backend.

## Deployment

This site is configured for Netlify deployment.

### Deploy with Netlify CLI

1. Install Netlify CLI:
```bash
npm install -g netlify-cli
```

2. Login to Netlify:
```bash
netlify login
```

3. Deploy:
```bash
netlify deploy --prod
```

### Deploy with Git

1. Push this repository to GitHub
2. Go to [Netlify](https://app.netlify.com)
3. Click "Add new site" → "Import an existing project"
4. Connect your GitHub repository
5. Netlify will automatically detect the configuration and deploy

## Features

- Responsive wedding RSVP form
- Guest dashboard to view responses
- Supabase backend for data storage
- Beautiful animations and design
